## Project Overview

GitHub Issue Tracker is a simple web application that allows users to view, search, and manage project issues.
Users can log in with admin credentials and explore all issues fetched from an API.

---

## Features

* 🔐 Admin Login System
* 📋 Display All Issues
* 📂 Filter Issues (All / Open / Closed)
* 🔎 Search Issues
* 📄 Issue Details Modal
* ⏳ Loading Spinner while fetching data
* 📱 Responsive Layout

Single Issue

Search Issue
https://phi-lab-server.vercel.app/api/v1/lab/issues/search?q={searchText}
